# 🏦 Bank Customer Churn Analysis Dashboard

A business-focused Power BI dashboard built to analyze customer churn, identify key churn drivers, and provide actionable recommendations to improve customer retention.

# 📌 Project Overview

Customer churn is one of the biggest challenges faced by banks. Losing existing customers increases acquisition costs and directly impacts revenue.

This project analyzes customer demographics, financial information, and customer behavior to answer four key business questions:

- How many customers are leaving?
- What factors contribute to customer churn?
- Which customer segments are at higher risk of leaving?
- What actions can the bank take to improve customer retention?

The dashboard is designed with storytelling in mind, helping decision-makers quickly understand churn patterns and identify areas requiring attention.

# 🎯 Business Problem

The bank wants to understand why customers leave and identify the factors contributing to churn so that effective retention strategies can be implemented.

# 📊 Dashboard Features

- Total Customers
- Active Members
- Churned Customers

### Customer Churn Analysis

- Churn by Geography
- Churn by Age Band

- Churn by Balance Tier
- Churn by Gender

### Interactive Features

- Pop-up slicer panel
- Key Influencer View
- Navigation between report pages
- Interactive filtering

# 💡 Key Business Insights

Some important findings from the analysis include:

- Germany has the highest customer churn among all regions.
- Inactive customers are significantly more likely to leave.
- Customers with only one banking product show higher churn.
- Customer demographics and financial characteristics influence churn differently across segments.
